## The Asset contains the following:
* Scalable Levels using `ENUM` and `STRUCT`
* Level unlocks as per the Player Progression
* Sophisticated Graphics with `Mobile Friendly Shaders`
* Simple Controls i.e. `Double Tap`
* Easily managable code using `Managers`
